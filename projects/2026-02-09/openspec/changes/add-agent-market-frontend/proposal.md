# Change: Add Agent Market Frontend

## Why
The frontend needs to align with the backend/contract roadmap to support agent discovery, role-aware access, and agent management flows.

## What Changes
- Add Agent Market frontend capabilities for listing, filtering, and viewing agents.
- Add role-aware login flow using wallet signature and optional role selection.
- Add agent creation/editing UI with tags and runtime/auth configuration fields.

## Impact
- Affected specs: agent-market-frontend (new)
- Affected code: apps/agent-market-fe (market, auth, agent management)
