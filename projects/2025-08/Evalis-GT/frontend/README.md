# Frontend

React + TypeScript + Vite frontend application for Evalis-GT.

## Development

```bash
npm install
npm run dev
```

## Building

```bash
npm run build
npm run preview
```

## Environment Variables

Create `.env.local` file:
```env
VITE_API_URL=http://localhost:5000
VITE_CLERK_PUBLISHABLE_KEY=your_clerk_key
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Clerk Authentication
- React Router
