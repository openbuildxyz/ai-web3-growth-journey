# 🚀 快速启动指南 - 阶跃星辰版

> **3 步配置，5 分钟启动！**

---

## ⚡ 超快速启动（3步）

### 第 1 步：获取 API Key（2分钟）

1. 访问: https://platform.stepfun.com/
2. 注册并实名认证
3. 创建 API Key，复制保存

### 第 2 步：配置项目（1分钟）

打开 `.env` 文件：
```bash
notepad c:\Users\郭岩\CodeBuddy\20260208111233\.env
```

填入配置：
```env
AI_PROVIDER=stepfun
STEPFUN_API_KEY=sk-你的实际Key
SEARCH_PROVIDER=baidu
```

保存文件（Ctrl + S）。

### 第 3 步：启动（1分钟）

```bash
# 双击运行
start-dev.bat

# 打开浏览器
http://localhost:5173
```

**完成！开始使用吧！** 🎉

## 📊 三种使用方式

### 方式 1: 本地使用（当前）
- **配置**: 仅需 API Key
- **访问**: http://localhost:5173
- **成本**: 0元（免费额度内）
- **适合**: 个人学习

### 方式 2: 局域网共享
```bash
# 1. 获取本机 IP
ipconfig
# 如: 192.168.1.100

# 2. 修改 client/.env
VITE_API_URL=http://192.168.1.100:3001/api

# 3. 其他人访问
http://192.168.1.100:5173
```
- **适合**: 团队协作

### 方式 3: 云端部署
```
Vercel (前端) + Railway (后端) + Supabase (数据库)
```
- **适合**: 随时随地访问
- **详见**: `阶跃星辰配置和部署指南.md`

## 🆘 遇到问题？

### API Key 错误
- 检查格式是否为 `sk-xxxx`
- 确认已实名认证
- 重启服务器

### 端口被占用
```bash
# 查找占用进程
netstat -ano | findstr :3001
netstat -ano | findstr :5173

# 结束进程
taskkill /PID <进程ID> /F
```

### 依赖安装失败
```bash
# 清除缓存重新安装
npm cache clean --force
npm install
```
