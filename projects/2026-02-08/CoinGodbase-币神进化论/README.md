## 🎯 核心概念

### 现状与冲突

5 个 AI Agent 作为独立链上基金 (DAO Funds)，各自持有真实资金，通过 API 在 Uniswap (DEX) 或 Binance (CEX) 上 24/7 自主交易。它们初始策略很强，但不够完美。

**冲突点**：用户在平台连接钱包进行自己的交易，系统实时监控用户 vs Agent 的收益率。

### 进化与反馈

当某个用户在特定周期（如 24 小时）内的收益率击败了某个 Agent：

1. **拜师** - Agent 会读取该用户的链上操作记录
2. **提取** - 分析策略特征
3. **修改** - 更新自己的交易代码

**回馈机制**：作为回报，该 Agent 未来产生利润的 **10%** 将自动分红给这位"人类导师"。

## 🤖 五大实盘 Agent

每个 Agent 都是一个 ERC-6551 (Token Bound Account) 钱包，拥有独立的资产和交易权限。

### No.1 巴菲特基金 (Agent-1)

| 属性 | 说明 |
|------|------|
| **策略** | 大盘定投 & 质押。只买 BTC/ETH，低买高卖，不做空 |
| **缺陷** | 熊市回撤大，资金利用率低 |
| **进化方向** | 如果被"做空玩家"击败，学会"套保"对冲 |
| **进化后** | Hedged Buffett (对冲巴菲特) |

### No.2 量化基金 (Agent-2)

| 属性 | 说明 |
|------|------|
| **策略** | DEX 链上套利，监控 Uniswap/Curve 价差搬砖 |
| **缺陷** | Gas 费飙升时会亏损，不懂宏观情绪 |
| **进化方向** | 学会"Gas 优化"或"暂停交易" |
| **进化后** | Smart Gas Quant (智能费率量化) |

### No.3 趋势基金 (Agent-3)

| 属性 | 说明 |

| **进化方向** | 学会在震荡区间"高抛低吸" |
| **进化后** | Adaptive Hunter (自适应猎手) |

### No.4 佛系指数 (Agent-4) - 市场基准

### No.5 巨鲸暗池 (Agent-5) - 终极 Alpha

## 🎮 玩法流程：从"竞争"到"共生"

### Step 1: 挑战 (Challenge)

- 用户在平台注册，绑定交易所 API 或钱包地址
- 正常进行交易（无需把钱给平台，平台只读取数据）
- **排行榜**：实时显示 User ROI vs Agent ROI
- **结算周期**：每日 UTC 0:00

### Step 2: 注入 (Injection) - 核心创新

**触发机制**：系统选出击败 Agent-X 的 Top 1 用户

**AI 学习过程**：
1. 抓取该用户的交易历史
2. 输入 LLM (GPT-4o) 分析交易特征
3. 生成优化代码，在沙盒中回测 30 天数据
4. 通过后实装上线

### Step 3: 获利 (Profit Sharing)

- **铸造权益**：策略被采纳的用户获得 "Strategy NFT"
- **智能分红**：
  - Agent 在接下来的 7 天内使用新策略产生的超额收益 (Alpha)
  - 智能合约自动计算 **10%** 利润
  - 通过流支付 (Superfluid) 实时打入用户钱包

> **Slogan: "Code once, earn forever." (一次操作，永久躺赚)**

## 🚀 快速开始

- Python >= 3.11
