## 什么是AI³ Growth Journey

「AI 正在吞噬软件，Web3 正在重构生产关系」，虽然 Vitalik 与 Sam Altman 还未在达沃斯握手，代码与智能合约却已经在重塑世界的运行规则。此刻，一场横跨亚太的开发者觉醒运动即将启程！

AI³ Growth Journey 是由 OpenBuild 与周周黑客松联合发起的 AI+Web3 系列活动。我们将从 3 月份开始，在未来的 10 个月里，每个月基于 AI + Web3 领域特定主题/生态系统展开线上共学，以及随机选择热点城市举办线下的 Mini-Hackathon，引导开发者产出与特定主题/生态相关的项目应用。

## 为什么AI与Web3的融合如此重要？

AI与Web3的融合正在创造全新的可能性：

- **降低技术门槛**：AI工具使编写复杂智能合约变得更简单
- **优化开发效率**：自动化代码生成、测试和审计流程
- **扩展应用场景**：实现自主代理、链上数据分析、个性化体验
- **赋能DAO治理**：提升协作效率和集体决策质量

这种融合为所有技术水平的参与者提供了进入机会，无论你是经验丰富的区块链开发者还是刚刚开始探索的新手。

## 两大受众群体

我们的内容专为两类开发者精心设计：

### 💡 对Web3不熟悉的开发者

如果你是AI工程师、Web2开发者或对区块链世界充满好奇，我们会帮助你：
- 理解区块链和智能合约的基础知识
- 使用AI工具快速掌握Solidity等Web3技术
- 构建你的第一个去中心化应用

你将学习：
- **理解Web3基础概念**: 区块链、智能合约、去中心化应用的核心原理
- **掌握基本工具**: 如何使用钱包、与区块链交互
- **应用AI编程助手**: 使用AI工具(如Cursor、Copilot)快速学习并编写智能合约
- **构建简单应用**: 从零开始构建你的第一个Web3应用

### 💎 Web3开发者寻求AI赋能

如果你已经熟悉区块链开发，但希望利用AI提升效率，我们将帮助你：
- 将AI代码助手集成到智能合约开发流程
- 优化复杂合约设计和审计流程
- 探索AI+Web3的创新应用场景

你将学习：
- **AI代码助手高级应用**: 将AI工具整合到智能合约开发工作流中
- **优化开发效率**: 用AI加速测试、审计和部署流程
- **突破技术瓶颈**: 解决复杂合约设计和优化问题
- **探索前沿应用**: 构建AI+Web3融合的创新项目

## 全年活动规划

在这个计划中，我们将每月探索不同的AI+Web3主题：

- **Write Smart Contract with AI Code Assistant** - 使用AI代码助手（如Cursor、Copilot）编写智能合约
- **How to write a professional AI Prompt** - 专业AI提示词工程
- **Build with Web3 AI Agent** - 构建Web3 AI代理（ai16z、virtuals等）
- **OnChain Data Insight with AI** - 使用AI分析链上数据
- **No code Dapps** - 无代码去中心化应用开发
- **AI Gaming / AIGC in Web3** - AI游戏和Web3中的AIGC

每个主题都包括线上Workshop和线下Mini-Hackathon，帮助你从理论到实践全面掌握技能。

## 三月主题: 使用AI代码助手编写智能合约

本月我们将集中探讨如何借助AI工具(如Cursor、Copilot)编写、测试和优化智能合约，无论你是Web3新手还是有经验的开发者，都能从以下内容中获益:

### 学习路径

1. **基础入门** (适合Web3新手)
   - 理解智能合约基本概念
   - 设置开发环境和AI助手
   - 使用AI编写第一个简单合约

2. **进阶实践** (适合有经验的开发者)
   - 用AI优化合约性能和安全性
   - 处理复杂业务逻辑的提示工程技巧
   - 结合AI进行合约测试和审计

3. **实战项目**
   - 从需求分析到部署全流程实践
   - 基于当前热点生态系统的应用开发
   - 准备参与深圳Mini-Hackathon
