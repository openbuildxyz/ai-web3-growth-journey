# StarQuest: AI增强的Web3探索游戏

> **团队规模：** 多人团队  
> **开发时长：** 4小时现场开发  

## 📖 项目简介

**StarQuest** 是一款规则上链、结果随机，并通过 AI 增强游戏趣味性的 Web3 游戏。游戏围绕"探索、选择、成长、奖励"展开，玩家将穿越由其他玩家创建的"星岛"，展开一次次充满变数与想象力的星愿旅程。每次选择都记录于链上，每次游戏过程都由 AI 生成独特的图文反馈，最终构成玩家的专属命运轨迹。

## ✨ 核心机制

### 🎯 链上游戏规则
- 所有探索路径选择、事件结果与奖励均由合约+VRF（链上可验证随机数）驱动
- 确保游戏公平性和公开透明性
- 防止作弊和操控

### 🤖 AI 游戏增强
- 每次游戏均由AI生成图文反馈
- 个性化剧情描述和场景图像
- 增强游戏沉浸感和叙事体验

### 📈 玩家成长系统
- 等级、技能、装备槽等成长体系
- 随游戏过程不断增强角色能力
- 解锁新技能和特殊效果

### 🏝️ 星岛成长系统
- 玩家创建的"星岛"也会随挑战次数成长
- 增加游戏的变化性和重复游玩价值
- 房主获得收益激励

### 🎁 资产留存
- 每次旅程的重大节点可生成"星愿纪念 NFT"
- 可展示、交易或参与后续活动
- 构建玩家的专属游戏历史

## 🎮 游戏玩法

### 1️⃣ 创建/进入星岛
- **房主模式**：支付费用创建星岛，自定义奖励资金池
- **探索模式**：选择星岛进行探索，支付入场费开始旅程
- **星愿描述**：基于输入生成星岛形象与事件风格

### 2️⃣ 多层探索与选择
- 每个星岛包含多层（通常为3~5层）
- 每层出现3个路径选项供玩家选择
- VRF返回对应事件类型：

| 事件类型 | 效果 |
|----------|------|
| 🌟 星辉奖励 | 获得 token 奖励与经验 |
| 🌀 星愿馈赠 | 获得 buff、技能道具或幸运祝福 |
| 💥 星屑干扰 | 探索失败，结束本局，入场门票被星岛收回 |

### 3️⃣ AI 图文生成
- **图像**：对应场景的AI生成图片
- **文本**：个性化剧情描述
- **记忆册**：串联为"星愿回忆录"，可mint成NFT

## 🧬 成长系统详解

### 玩家成长
| 内容 | 描述 |
|------|------|
| 🎖 等级 | 每次探索获得 EXP，提升等级解锁新技能位 |
| 🧠 技能 | 被动技能，如"免疫首次失败"、"重新掷一次命运" |
| 💠 星纹 | 可装备的 NFT 组件，提升幸运、事件收益 |
| 📜 星愿纪念 | 每次完整旅程生成图文结合的纪念 NFT |

### 星岛成长
| 内容 | 描述 |
|------|------|
| 📈 星岛等级 | 每被挑战一次成长 +1，提升奖励上限 |
| 🏷️ 星岛标签 | AI自动标注类型，如「梦幻型」、「高风险型」 |
| 🎨 星岛进化 | 达到等级阈值解锁新的AI模板与图像风格 |
| 💰 收益机制 | 玩家失败时入场费归属房主资金池 |

## 💰 游戏经济

| 动作 | 成本 | 回报 |
|------|------|------|
| 创建星岛 | 0.001 ETH + 奖池（可自定） | 解锁星岛，成长回报、分成收益 |
| 探索一次 | 0.0005 ETH | 成功可获星辉 + 星岛奖励，失败门票归星岛 |
| 通关奖励 | 无额外成本 | 可赢得星岛资金池中一部分奖励 |
| 纪念NFT铸造 | 0.00001 ETH（可选） | 生成 AI 图+文 旅程记录 NFT |

## 🎲 示例游戏流程

1. 玩家 Lv3，进入「星岛·晨光之丘」（房主注入 0.01 ETH 奖池）
2. 第1层选择"微星阶" → 触发奖励事件
   - AI图像：金色石阶闪耀星辉
   - 文本描述："你的第一步，照亮了心中最柔软的愿望。"
3. 第3层可能触发失败事件 → 退出探索，入场费归星岛
4. 若通关成功：获得 0.003 ETH 奖励 + 经验 + 星辉 + 纪念 NFT
