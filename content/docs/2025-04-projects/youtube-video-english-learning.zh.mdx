# 🌱 YouTube Video English Learning - 英语学习插件

## 🌟 项目概述

YouTube Video English Learning是一个基于链上认证、质押激励的英语学习插件，专为Web3从业者设计的Chrome浏览器扩展。该项目旨在解决中国区Web3相关人员走向国际化过程中面临的英语交流障碍，通过在观看YouTube视频过程中记录生词、回顾词汇、增强听力理解能力，帮助用户达到听力与词汇量双重提升的效果。

### 💡 核心创新

- **实时词汇记录**：在观看YouTube视频时遇到生词可以即时记录
- **时间戳功能**：自动记录视频时间戳，方便后续回播复习  
- **链上认证**：基于区块链的学习成果认证
- **质押激励**：通过质押机制激励持续学习

## 🛠️ 技术实现

### 核心技术栈
- **Chrome Extension API**：利用浏览器扩展API构建插件基础架构
- **JavaScript**：前端逻辑实现
- **HTML/CSS**：用户界面设计
- **Content Script**：与YouTube页面交互
- **Background Script**：后台数据处理
- **Web3集成**：钱包连接和区块链交互

### 主要功能模块
1. **Content Script**：负责在YouTube页面中注入功能，监听用户操作
2. **Popup Interface**：主要的用户交互界面
3. **Options Page**：插件设置和配置页面
4. **Background Service**：后台服务处理
5. **Web3 Integration**：区块链功能集成

### 数据存储
- Chrome Extension Storage API进行本地数据存储
- 区块链钱包集成实现去中心化存储
- IPFS分布式存储支持

## 🎯 解决的问题

英语正常交流能力已成为中国区Web3相关人员走向国际化的最大障碍。传统的英语学习方式往往脱离实际应用场景，而Web3行业的专业词汇和表达方式更是需要在真实的视频内容中学习。本项目从英语听力这个基础方向出发，为用户提供了一个在真实语境中学习英语的工具。

## 🎮 使用方法

1. 在Chrome浏览器中安装扩展
2. 连接Web3钱包
3. 打开YouTube视频
4. 遇到生词时点击扩展图标或使用快捷键
5. 输入生词并自动记录当前时间戳
6. 在扩展管理界面中复习已记录的词汇
7. 通过时间戳功能回到原视频位置加深记忆
8. 完成学习任务获得链上认证
