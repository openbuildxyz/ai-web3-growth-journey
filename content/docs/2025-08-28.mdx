# AI+Web3 Mini-Hackathon Event Report

## Event Overview
The AI+Web3 Mini-Hackathon was successfully held as a key component of the AI³ Growth Journey, a collaborative initiative by **OpenBuild** and **Weekly Hackathon**. The event focused on driving innovation at the intersection of Artificial Intelligence (AI) and Web3 technologies, aiming to equip developers with the skills to build the next generation of decentralized applications (dApps) and user experiences.

## Event Details
| Item | Details |
| --- | --- |
| **Event Name** | AI+Web3 Mini-Hackathon |
| **Vision** | To promote collaborative innovation in AI and Web3, empowering developers and entrepreneurs to build AI+Web3 applications. |
| **Organizers** | OpenBuild, Weekly Hackathon |
| **Venue** | 301ALH, Bennett University, Noida |
| **Format** | Offline Mini-Hackathon (RSVP-based for selected students) |
| **Focus** | Developing innovative AI+Web3 applications (e.g., AI-powered dApps, smart contract automations, AI-driven Web3 UX). |

## Event Objectives
The primary goal of the Mini-Hackathon was to provide a hands-on environment for participants to seize the initiative in the AI+Web3 wave. By tackling real-world challenges, participants were expected to gain unique advantages and practical experience in integrating these two transformative technologies.

## Suggested Problem Statements
The hackathon challenged participants to address critical needs within the Web3 ecosystem by leveraging AI:
- Building an AI-Powered Smart Contract Auditor
- Developing a drag-and-drop AI interface for launching a DAO
- Creating an AI system to check and flag fake NFTs by cross-referencing on-chain metadata
- Building an AI assistant that converts legal clauses into Solidity contract logic
- Creating an AI tool for gas-efficient Solidity contract rewrites with usage comparison
- Developing an AI agent for real-time wallet monitoring and warning against suspicious transactions/approvals
- Building a decentralized arbitration system where AI translates natural language disputes into on-chain contract logic
