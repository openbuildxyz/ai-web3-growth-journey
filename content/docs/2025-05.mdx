# Explore the On-Chain World with AI Mini Hackathon

## 🌟 活动愿景

AI 和 Web3 作为引领下一代技术革命的核心驱动力，二者的深度融合将重塑未来的技术生态与商业模式。对此，OpenBuild 与 周周黑客松联合发起 **AI³ Growth Journey**，旨在推动 AI 与 Web3 技术的协同创新，赋能开发者与创业者。

AI³ Growth Journey 计划每月在亚洲的主要地区（中国、泰国、越南、马来西亚、新加坡、印度等）举办线上 AI+Web3 Workshop 和一次线下 Mini-Hackathon，帮助更多开发者了解和学习 AI+Web3，并以构建 AI+Web3 应用为最终目标。

## 项目提交指南

### GitHub代码提交流程

所有参赛项目必须通过Pull Request提交到官方仓库: [https://github.com/openbuildxyz/ai-web3-growth-journey](https://github.com/openbuildxyz/ai-web3-growth-journey)

具体步骤如下:

1. **Fork仓库**: 访问[ai-web3-growth-journey](https://github.com/openbuildxyz/ai-web3-growth-journey)仓库并点击右上角的"Fork"按钮

2. **克隆Fork后的仓库**:
   ```bash
   git clone https://github.com/你的用户名/ai-web3-growth-journey.git
   cd ai-web3-growth-journey
   ```

3. **创建新分支**:
   ```bash
   git checkout -b 团队名-project
   ```

4. **添加项目文件**:
   - 在`projects/2025-05/`目录下创建以你的团队名命名的文件夹
   - 文件夹内应包含:
     - `README.md`: 项目介绍、团队信息、技术实现等
     - `contracts/`: 智能合约代码
     - `frontend/`(可选): 前端代码
     - `demo.md`: 包含Demo链接和截图
     - `prompts.md`: 记录与AI助手的关键提示词交互

5. **提交更改**:
   ```bash
   git add .
   git commit -m "Add 团队名 project for May 2025 Hackathon"
   git push origin 团队名-project
   ```

6. **创建Pull Request**:
   - 访问你Fork的仓库
   - 点击"Compare & pull request"按钮
   - 填写PR描述，包含项目名称、团队成员、项目简介和技术亮点
   - 提交PR

### 提交内容要求

- **代码注释**: 必须包含详细注释，特别说明AI生成的部分
- **AI提示记录**: 记录与AI的关键交互提示词，展示AI如何帮助完成项目
- **项目文档**: 必须包含完整的README.md说明项目目的、技术实现和使用方法
- **展示页面**: 必须创建展示页面，包含项目概述、技术实现、AI辅助过程和团队收获

### 截止时间

所有Pull Request必须在**演示前提交完成**。
