# Trisolaris

## 项目概述

Trisolaris 是一个基于区块链的卡牌抽卡游戏，玩家可以使用 TRI 代币抽取不同稀有度的卡牌，并有机会获得三连奖励。项目结合了加密货币的概念和三连（Triad）机制，为玩家提供了一个既有趣又有收益的游戏体验。

## 核心功能

1. **代币经济系统**
   - TRI 代币作为游戏内货币
   - 抽卡消耗代币
   - 三连奖励机制返还代币

2. **卡牌抽取系统**
   - 四种稀有度的卡牌（普通、稀有、史诗、传说）
   - 基于 Chainlink VRF 的可验证随机抽卡
   - 不同稀有度卡牌对应不同概率和奖励倍数

3. **三连奖励机制**
   - 连续抽到三张相同卡牌触发奖励
   - 奖励基于卡牌稀有度计算
   - 传说卡三连可获得奖池奖励

4. **奖池系统**
   - 每次抽卡贡献一部分代币到奖池
   - 传说卡三连可获得50%奖池奖励
   - 动态增长的奖励机制

## 技术架构

1. **智能合约**
   - Solidity 编写的游戏核心逻辑
   - 基于 OpenZeppelin 的安全合约库
   - Chainlink VRF 提供可验证的随机性

2. **前端应用**
   - React.js 构建的用户界面
   - ethers.js 实现区块链交互
   - 响应式设计适配多种设备

3. **区块链集成**
   - 支持 Polygon 主网和 Mumbai 测试网
   - MetaMask 钱包集成
   - 高效的链上交互设计

## 项目亮点

1. **创新游戏机制**
   - 结合传统抽卡游戏与区块链技术
   - 三连机制增加游戏策略性
   - 可验证的随机性保证公平性

2. **经济模型设计**
   - 可持续的代币循环系统
   - 基于稀有度的差异化奖励
   - 奖池机制增加长期参与动力

3. **用户体验优化**
   - 简洁直观的界面设计
   - 低门槛的游戏参与方式
   - 实时反馈的抽卡体验

## 未来展望

1. **游戏扩展**
   - 增加更多卡牌类型和稀有度
   - 引入卡牌合成和升级系统
   - 开发卡牌交易市场

2. **社区建设**
   - 建立玩家社区和排行榜
   - 举办定期抽卡活动和比赛
   - 引入社区治理机制

3. **跨链支持**
   - 扩展到更多区块链网络
   - 支持跨链资产转移
   - 多链奖池互通
